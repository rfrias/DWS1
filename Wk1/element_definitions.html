HTML Element Definitions

Submitted by: Ramon Rafael G. Frias IV

<p>
Used to create a paragraph
<b>
Used to makes words appear bold
<i>
Used to makes words appear italic
<sup>
Used to makes characters appear as a subscript i.e. mathematical concepts
<sub>
Used to makes characters appear as a subscript i.e. chemical formulas
<br />
Used to add a line break in the middle of a paragraph
<hr />
Used to add break between themes
<h1>
Used to make a main heading
<h2>
Used to make a sub heading level 1
<h3>
Used to make a sub heading level 2
<h4>
Used to make a sub heading level 3
<h5>
Used to make a sub heading level 4
<h6>
Used to make a sub heading level 5
<strong>
Used to show strong importance
<em>
Used to show emphasis
<blockquote>
Used to indent longer quotes
<q>
Used for short quotes
<abbr>
Used on abbreviations or acronyms
<cite>
Used to reference a piece of work i.e. book
<dfn>
Used when chosen terminology is used for the first time
<address>
Used to contain details of author(s) that is/are used
<ins>
Used to show content that has been used into a document by underlying the word
<del>
Used to show text that has been deleted in a document by putting a ling across the word
<s>
Shows something that is irrelevant by putting a line across the word
<ol>
Used to create ordered list by using numbers
<li>
Used to indicate listed item
<ul>
Used to create unordered list by using bulletins
<dl>
Used when listing terms with definitions
<dt>
Used to contain the term that is being defined
<dd>
Used to contain the definition of the term being defined
